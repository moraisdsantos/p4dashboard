# Métodos de Escuta Cidadã (Vite + React + Tailwind v4)

## Rodar localmente

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

### Dados (CSVs)
Os CSVs iniciais estão em `src/imports/`:
- `ficha_tecnica.csv`
- `passos_aplicacao.csv`

Você também pode carregar novos CSVs pela interface.
